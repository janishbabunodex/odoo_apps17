📊 Inventory FSN Report

Classify your stock as Fast, Slow, or Non-Moving — with PDF & Excel export built in.

Odoo 17 Inventory / Stock Free & Open Source LGPL-3

Overview

Inventory FSN Report brings classic FSN (Fast–Slow–Non-moving) inventory analysis directly into Odoo 17. Using actual stock movement history, it calculates the turnover ratio for every product/location combination and classifies each item so your procurement and warehouse teams can act on real data — not guesswork.

Features

📅 Flexible Date Range

Analyse any period — weekly, monthly, quarterly or custom — with simple Start/End date pickers.

🔍 Powerful Filters

Narrow results by specific Products, Product Categories, Companies, and Warehouses.

🏷️ FSN Type Filter

Show All items or drill into Fast, Slow, or Non-Moving items instantly from the wizard.

📋 Colour-Coded List View

Green badge for Fast, amber for Slow, red for Non-Moving — at a glance in the tree view.

📈 Graph & Pivot Views

Bar chart grouped by FSN classification plus a pivot table for category-level analysis.

📄 PDF Export

Professional QWeb PDF report with header, data table, and colour-coded FSN legend.

📊 Excel Export

Formatted .xlsx file with title, period subtitle, colour-coded FSN cells, and numeric formatting.

🏢 Multi-Company Ready

Filter by one or more companies; warehouse selection automatically scopes the internal locations.

FSN Classification Rules

ClassLabelTurnover Ratio ConditionInterpretation
F Fast Moving Turnover Ratio > 3 High-velocity items — ensure adequate stock levels
S Slow Moving Turnover Ratio between 1 and 3 Moderate demand — review reorder points
N Non Moving Turnover Ratio ≤ 1 Dead stock — consider clearance or write-off

Formula: Turnover Ratio = Sale Qty ÷ Average Stock  |  Average Stock = (Opening Stock + Closing Stock) / 2

How It Works

  1. Go to Inventory → Reporting → FSN Report.
  2. Set the Start Date and End Date for the analysis period.
  3. Optionally select the FSN Type, products, categories, companies, or warehouses.
  4. Click View Report to open the interactive tree + graph view inside Odoo.
  5. Click PDF to download a formatted print-ready report.
  6. Click Excel to download a colour-coded spreadsheet.

Installation

  1. Download and unzip into your Odoo addons directory.
  2. Restart the Odoo server.
  3. Enable Developer Mode.
  4. Go to Apps → Update App List.
  5. Search for Inventory FSN Report and click Install.
